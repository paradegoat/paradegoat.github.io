SafeSpace

Working with a team for our final project at The Iron Yard, SafeSpace is a mobile app for emptath360 and Luv is a Gift. It allows users access quick help if in an abusive situation and gives them the ability to log their experiences in a diary for possible legal battles. The user is able to enter information about their environment and description and picture of perpetrator. They also have the option of a one touch call to 911 or to a safety contact if immediate police help is not needed. I was tasked with building the database with MongoDB and mLab. The project was built using Node.js. We worked on Github using personal development branches pushed to shared development branch and finally deployed the master on Heroku. check it out below: username:guest password: Password1
